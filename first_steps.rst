First Steps
===========

Setting Up Cuneiform
--------------------

Requirements
^^^^^^^^^^^^

- Linux, BSD, or Mac OS Operating System
- Erlang/OTP 19.0 or higher
- Rebar 3.0 or higher

Download
^^^^^^^^

    git clone https://github.com/joergen7/cuneiform.git
    cd cuneiform
  
Build
^^^^^

    rebar3 escriptize

Install
^^^^^^^

    cp _build/default/bin/cfl ~/bin

Starting Cuneiform
------------------

    cfl
